## Description

Demo project using [https://swapi.dev/](https://swapi.dev/)

Built using:

- Create React App
- Chakra UI
- React Testing Library
- Cypress

## Commands

`yarn start` to run locally

`yarn build` to build

`yarn test` to run tests

`yarn cypress:run` to run cypress tests
